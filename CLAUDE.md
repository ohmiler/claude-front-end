# CLAUDE.md

## Project Goal
Build a professional 5-page front-end website for an AI-era software development company called "Astra AI Software Studio".

## Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS
- React Router
- lucide-react for icons

## Pages
1. Home
2. Services
3. Case Studies
4. Process
5. Contact

## UI Requirements
- Modern software company aesthetic
- Dark, premium, AI-native visual style
- Responsive design for mobile, tablet, desktop
- Reusable components
- Semantic HTML
- Accessible buttons, labels, and navigation
- Add data-testid attributes for testable UI elements

## Commands
- npm run dev
- npm run build
- npm run lint

## Quality Rules
- Do not put all code in App.tsx
- Create reusable components
- Keep page content in structured data files when possible
- Run build after major changes
- Fix root causes, do not silence errors
